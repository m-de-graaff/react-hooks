# React Hooks Playground

Interactive playground for previewing and testing React hooks from `@m-de-graaff/react-hooks`.

## Development

```bash
cd playground
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Deploy to GitHub Pages

```bash
pnpm deploy
```

This will build the app and deploy it to the `gh-pages` branch.
